# User Management API

## Features
- CRUD operations (Create, Read, Update, Delete)
- Input validation using Data Annotations
- Middleware for request logging
- Built using ASP.NET Core Web API

## API Endpoints
- GET /api/users
- GET /api/users/{id}
- POST /api/users
- PUT /api/users/{id}
- DELETE /api/users/{id}

## Tools Used
- .NET Web API
- GitHub Copilot
- Swagger for testing
